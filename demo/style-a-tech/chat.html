A

Based on the work order database, this issue is related to IBMS Integration configuration. Here are the common causes and solutions:

Possible Causes:

  1. Movement type 101 not configured in SAP
  2. Missing authorization for the user
  3. Plant/Storage location mapping issue

Recommended Steps:

  1. Check SAP Transaction OMJJ for movement type 101
  2. Verify user roles in SU01
  3. Confirm plant mapping in Dmall settings
