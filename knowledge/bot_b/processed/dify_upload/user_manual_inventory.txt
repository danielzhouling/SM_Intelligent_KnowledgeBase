[Receiving detail information Instruction:]

Receiving detail information Instruction:

Receive Qty must be equal to or less than Plan Qty.
If Receive Qty is equal to Plan Qty, discrepancy type could be No differ, Over served, Damaged or Expiry.
If Receive Qty is less than Plan Qty, discrepancy type could be Underserved or Wrong served.
If discrepancy type is Over served, Damaged or Expiry, discrepancy qty should be input by users.
If discrepancy type is Underserved or Wrong served, discrepancy qty will be auto calculated and shown.
If discrepancy type is No differ, discrepancy qty is default as 0.
Profit center must be input if discrepancy type is underserved, wrong served or Damaged.
