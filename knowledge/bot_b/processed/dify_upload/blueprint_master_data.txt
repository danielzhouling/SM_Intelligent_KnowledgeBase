[3. Function Design Solution > 3.1 Site Master data system design and structure > 3.1.3 Inventory location by site]

1.AS-IS Overview

Inventory sites are managed in SAP

2.To-Be Overview

Inventory sites will be managed in DMALL system, total 7 inventory sites

3.To-Be Flow

In the DMALL system, the store will be divided into 7 inventory locations. And each store operation function/ movement action will operate in a specific inventory location. Each inventory location has its own inventory data. It will not be mixed up together so that we can clearly identify the current inventory situation and make smart decisions.

SLOC 0001 Selling Area: selling area inventory indicates the sales floor goods quantity. When POS selling, inventory should be deducted from the selling area.

SLOC 0025 Dark Inventory: This location is used for storing goods for online order picking. And the location is on the sales floor.

SLOC 0031 Backroom: Backroom inventory indicates the quantity of storage goods when goods are replenished into sales floor shelves, users should perform the stock transfer function to adjust the inventory

SLOC 0002 Offsite Warehouse: This location indicates the storage goods quantity in the outside warehouse

SLOC 0007 STDR from DC&Store: claim location is only used for delivery discrepancy report scenario. these stock should not be sellable or used for any business. Stores should separate an individual area for displaying these goods.

SLOC 0003 Bad Order Location: bad order location is only used for return scenario.

SLOC 0024 Holding location. These inventories should not be sellable or used for any business.store should separate an individual area for displaying these goods.

SLOC0001 selling area, SLOC 0025 Dark Inventory, SLOC 0031 backroom will be applied to picking system.

Inventory location can be set up by store level

Inventory site to be maintained in advance.

4. TO-BE Description

Create inventory site and fill in necessary information
Choose existing inventory sites to be applied when maintaining the new site
