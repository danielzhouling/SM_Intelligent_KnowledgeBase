7. Merchandising Label Management – Label View
Operation steps:

Merchandising Label source:a. Preset (Pre-designed label which cannot be edit manually)b. Manual (created by user)

User can have a whole view of all labels (even created by others) Also can filter out the label by the name or status

Data Dashboard
7. Merchandising Label Management – Preset Label Article (1)
Operation steps:

Can use label filter to type and find the label article list immediately

Article in some preset labels (top_N, SC/PWD etc.) are integrated from SAP which means can not be edited. Users only have the view authority.

Data Dashboard
7. Merchandising Label Management – Preset Label Article (2)
Operation steps:

Article in other preset labels (Hardline MBU etc.) are manually maintained by users which can be changed from Dmall label system directly

Data Dashboard
7. Merchandising Label Management – Create Label
Operation steps:

Click the [Create] button to start manual label set up
Fill up the information a. Label Nameb. Label Code (can auto generate)c. Entity Level (by article or by site-article)d. Valid period button (limit the article validation in this label or not)e. Description

Data Dashboard
7. Merchandising Label Management – Article Bind for create label
Operation steps:

Find the create label in view page, it will display the created information.

Next step is to bind the article for this label, click [Article Bind] button.Also user can edit or disable this label according to the business requirement.

Data Dashboard
7. Merchandising Label Management – Article Bind for create label
Operation steps:

User can choose import the article list or manually add the article for this label

After article bind, user can go back this page to view the article list.

Data Dashboard
7. Merchandising Label Management – Label influence for reports
Operation steps:

User can see the label performance in related reports immediately
