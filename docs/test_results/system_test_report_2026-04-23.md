## 三、失败原因分类

### A. Bot 未配置 Dify API Key（6个用例，预期失败）

Bot 当前状态为 `draft`（未配置 Dify API Key），导致用户端无法进入聊天页。以下用例在 Bot 配置完成后应通过：

- TC-U-07 选择Bot进入聊天
- TC-U-09 发送消息基本对话
- TC-U-18 提交"有用"反馈
- TC-U-19 提交"没用"反馈+原因
- TC-U-20 反馈按钮状态验证
- TC-U-21 重复反馈拦截

**解决方案**: 在管理后台为 Bot A 和 Bot C 配置 Dify API Key

### B. 前端 Bug（4个用例，需修复代码）

| Bug | 影响用例 | 严重程度 |
|-----|---------|---------|
| `AdminData` 未定义，admin 页面 JS 报错 | TC-A-03, TC-A-04, TC-A-09 | 🔴 严重 |
| 非管理员可登录管理后台 | TC-A-02 | 🟡 中等 |
| Bot 权限过滤未生效 | TC-U-05a, TC-U-05b | 🟡 中等 |
